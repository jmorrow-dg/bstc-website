Our highest-rated event to date. 35 builders spent an afternoon getting hands-on with AI automation.

**Format:** Talk (30 min) + Live Demo (30 min) + Hands-On Building (30 min)

**What was covered:**
- How to think about AI automation (what to automate vs what to keep human)
- Live build of an AI-powered lead qualification pipeline using n8n + Claude API
- Hands-on segment where attendees built their first n8n workflow

**Results:**
- 15 attendees successfully built and tested their first n8n workflow
- 3 founders committed to implementing AI automation in their businesses within the next week
- Multiple requests for a follow-up session on advanced workflows

**Feedback score:** 4.8 / 5.0 (highest ever)
